# Localisation Indoor PFE

Projet Unity de localisation intérieure pour le PFE.

## Structure du projet

- `Assets/` : ressources Unity, scènes et matériaux.
- `Packages/` : configuration des packages Unity.
- `ProjectSettings/` : paramètres du projet Unity.

## Utilisation

1. Ouvrir Unity Hub.
2. Ajouter le dossier du projet `localisation Indoor`.
3. Ouvrir le projet dans Unity.

## Git

Ce dépôt contient un `.gitignore` Unity pour éviter de versionner les fichiers générés.
