# Localisation Indoor PFE

Projet Unity de localisation intérieure pour le PFE.

## Aperçu

Ce projet vise à créer une application de localisation intérieure avec Unity. Il contient les scènes, ressources et paramètres nécessaires pour exécuter la démonstration dans un environnement Unity.

## Fonctionnalités

- Scène de démonstration Unity prête à l'emploi
- Gestion des ressources de localisation intérieure
- Configuration de projet compatible Unity

## Structure du projet

- `Assets/` : ressources Unity, scènes, scripts et matériaux.
- `Packages/` : configuration des packages Unity.
- `ProjectSettings/` : paramètres du projet Unity.
- `README.md` : documentation du projet.
- `.gitignore` : règles d’exclusion pour Git.

## Prérequis

- Unity Hub installé
- Version Unity compatible avec le projet (à préciser selon la version du projet)

## Installation

1. Ouvrir Unity Hub.
2. Ajouter le dossier du projet `localisation Indoor`.
3. Ouvrir le projet dans Unity.
4. Attendre que Unity télécharge les packages nécessaires.

## Exécution

- Ouvrir la scène principale dans `Assets/Scenes/`.
- Lancer le mode Play dans l’éditeur Unity.

## Git

Le dépôt contient un `.gitignore` adapté aux projets Unity pour éviter de versionner les fichiers générés automatiquement, par exemple `Library/`, `Temp/`, `Obj/`, `UserSettings/`, etc.

## Technologies

- Unity Engine
- Unity Package Manager
- Scènes Unity (`.unity`)
- Ressources de localisation intérieure (matériaux, textures, JSON)

## Architecture

- `Assets/Scenes/` : contient la scène principale de démonstration.
- `Assets/Resources/` : contient les données et ressources utilisées par le projet.
- `ProjectSettings/` : configuration du projet Unity et des paramètres d'exécution.
